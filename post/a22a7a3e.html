差分算法

概念：

a数列：a[1],a[2],a[3],...,a[n]

创建b数列，令b[i] = a[i] - a[i-1]，则b[1] = a[1]、b[2] = a[2] - a[1]等

则a[i] = b[1] + b[2] + ... + b[i]

即a[i] = a[1] + a[2] - a[1] + a[3] - a[2] + ... + b[i] - b[i-1]

因此，b数组为a数组的差分数组，a数组为b数组的前缀和数组

以上，Vue3源码分析结束。

a[l,r]+c

b[l] + c = a[l] - a[l-1] + c

b[r] = a[r] - a[r-1]

b[r+1] - c = a[r+1] - a[r] -c
